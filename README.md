# AIGenesis
AIGenesis is an AI-powered platform that converts text prompts into professional-quality videos

AIGenesis is a full-stack AI-driven video generation platform built using ReactJS, Vue.js, FastAPI, Python, PostgreSQL, and Tailwind CSS. This final year project automates the creation of professional-quality videos from simple text prompts. It integrates OpenAI for intelligent script generation, Google Text-to-Speech (GTTS) for lifelike audio synthesis, and the Pexels API for sourcing high-quality stock footage.
